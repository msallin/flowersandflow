# FlowersAndFlow

A community club dedicated to **Poi** — the Māori performance art of swinging weighted objects through rhythmic patterns.

## About

FlowersAndFlow is a welcoming space for flow artists of all levels. We host weekly sessions, beginner workshops, and performances throughout the year.

Static single-page site hosted on [GitHub Pages](https://pages.github.com/). Open `index.html` to view locally.
